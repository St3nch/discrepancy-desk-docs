# Automation Boundaries

## Allowed

- drafting
- summarizing
- source extraction
- claim classification
- context retrieval
- manual checklist generation
- metrics analysis
- manual-ready post preparation

## Not Allowed

- autonomous posting
- autonomous replying
- automatic likes
- automatic follows
- automatic reposts
- automatic DMs
- keyword-triggered replies
- engagement pod behavior
- coordinated multi-account amplification

## Manual Posting First

Approved content becomes manual-ready.

User manually posts it.

User then records the published URL.

## Future Write API

Any future write API requires a separate explicit approval decision and new safety design.

It is not part of early MVP.
