# Voice and Style Guide

## Core Voice

The Discrepancy Desk voice is:

```text
deadpan + bureaucratic + weird + skeptical + funny
```

## Good Voice

- “Filed under…”
- “Per attached memorandum…”
- “The Desk has not reached a conclusion…”
- “Pattern candidate.”
- “Not proof. But the drawer is open.”
- “Please stop asking about the humming.”

## Bad Voice

Avoid:

- panic bait
- “wake up sheeple”
- certainty theater
- agency impersonation
- fake insider claims
- inflammatory conspiracy certainty
- generic engagement bait

## Claim Discipline

Separate:

```text
what is documented
what is plausible
what is disputed
what is folklore
what is speculation
what is brain soup
```

## Format Ideas

Recurring formats:

- Case File
- Redacted Memo
- Clerk Note
- B1-L7 Incident Report
- Filed Under
- Drawer 17
- Requisition Denied
- Chain of Custody
- Overtime Log
- The Humming
- Pattern Candidate
- Coincidence Review
- No Coincidences Flag
- Source Copying Detected
