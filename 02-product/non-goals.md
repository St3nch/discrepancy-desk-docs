# Non-Goals

## Not Building Early

- autonomous posting
- autonomous replying
- AI reply bot
- scheduler
- X write worker
- engagement automation
- likes/follows/reposts automation
- React SPA
- Postgres
- Redis/Celery/job queues
- vector DB at start
- multi-agent orchestration
- multi-account UI
- complex analytics/charts before data
- ML best-time-to-post
- thread builder
- auto screenshots
- perfect DOM parser
- growth-hacking dashboard
- full archive API backfill

## Not Becoming

- fake government account
- classified leak account
- conspiracy certainty machine
- engagement pod
- spam network
- generic social media dashboard

## Doctrine

Do not build the machine that gets the account banned before the account has a personality.
